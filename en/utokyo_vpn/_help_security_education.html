Troubleshooting guide: When the error message “Sorry, but we’re having trouble signing you in.” appears There is a possibility that you have not passed the information security education confirmation test, which is required to use UTokyo VPN. Please check the page for the information security course, and complete and pass the confirmation test. Once you pass the test, the results will be immediately reflected in the system.
If the issue is still not resolved, please consult the Technical Support Desk.
